\section{Correction de bugs} \label{act3}
\subsection*{But de l'activit\'{e}}
Correction de bugs.
\subsection*{Description}
La liste complète est dans le site diamantBug, où les bugs sont
classés comme amélioration, ou bug. Un bug peut être un faux bug,
pour cette raison il ne va pas se trouver dans les bugs corrigés.

Comme le bugs sont corrigés en urgence et un rapport existe dans
diamantBug nous ne donnons pas la durée du travail de correction.

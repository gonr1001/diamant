\subsection{Réorganisation suite aux nouvelles structures (fini 3 déc 04)} \label{reorgStruc1}\paragraph{But de la sous-activit\'{e}}Réorganisation de packages, suite à la nouvelle structure.\paragraph{Description}Voici les changements faits :\begin{description}  \item[\verb!dConstant!] Contient \verb!DConst! et  \verb!DStringFrRes!, cette dernière contient les chaînes en  Fran\c{c}ais.  \item[\verb!dDeveloper!] Contient les classes qui ne sont pas  inclues avec la version livrable.  \item[\verb!dInterface!] Contient les classes  qui affichent  quelque  chose à l'écran.    \begin{description}    \item[\verb!x!] à définir.    \item[\verb!y!] à définir.    \item[\verb!z!] à définir.    \end{description}  \item[\verb!dInternal!]    \begin{description}    \item Contient classe \verb!DModel! contenant les données de    l'application    \item[\verb!dData!] Contient les classes  correspondantes à    chaque ensemble de données.    \item[\verb!dDataXML!] Contient les classes où on traite les    données XML de l'extérieur. Va disparaître.    \item[\verb!dDataTxt!] Contient les classes où on traite les    données textuelles de l'extérieur. Va disparaître.    \item[\verb!dOptimization!] Contient les classes où on traite    l'optimisation : algorithmes ou conditions ou structures    auxiliaires.    \item[\verb!dTimeTable!] Contient les classes nécessaires à la    grille horaire.    \item[\verb!dUtil!] Contient les classes génériques, mais nécessaires uniquement à    \diamant{}, utilisées    dans \verb!dInternal!    \end{description}  \item[\verb!dmains!] Contient les \verb!main!s pour lancer les  différentes applications.  \item[\verb!dTest!]\end{description}

\subsection{Réorganisation 1 (fini 10 sep 04)} \label{reorg1}\paragraph{But de la sous-activit\'{e}}Réorganisation de packages\paragraph{Description}Voici les changements faits :\begin{description}  \item[\verb!dConstant!] Contient \verb!DConst! et  \verb!DStringFrRes!, cette dernière contient les chaînes en  Fran\c{c}ais.  \item[\verb!dDeveloper!] Contient les classes qui ne sont pas  inclues avec la version livrable.  \item[\verb!dInterface!] Contient les classes  qui affichent  quelque  chose à l'écran.    \begin{description}    \item[\verb!x!] à définir.    \item[\verb!y!] à définir.    \item[\verb!z!] à définir.    \end{description}  \item[\verb!dInternal!]    \begin{description}    \item Contient classe \verb!DModel! contenant les données de    l'application    \item[\verb!dDataTxt!] Contient les classes où on traite les    données textuelles de l'extérieur.    \item[\verb!dDataXML!] Contient les classes où on traite les    données XML de l'extérieur.    \item[\verb!dOptimization!] Contient les classes où on traite    l'optimisation : algorithmes ou conditions ou structures    auxiliaires.    \item[\verb!dTimeTable!] Contient les classes nécessaires à la    grille horaire.    \item[\verb!dUtil!] Contient les classes génériques, mais nécessaires uniquement à    \diamant{}, utilisées    dans \verb!dInternal!    \end{description}  \item[\verb!dmains!] Contient les \verb!main!s pour lancer les  différentes applications.  \item[\verb!dTest!]\end{description}Une librairie unique des classes génériques regroupant entreautres les libraries \verb!theLibrary! et \verb!xmlLibrary! plusd'autres classes non dépendantes de \diamant{}. Tout sur unpackage qui fera un \pro{.jar} indépendant.

\chapter*{Checkout List pour une classe}



\begin{enumerate}
    \item commentaire général bien fait et à la bonne place.
    \item commentaire général avec le nom de la classe.
    \item bon nom du package.
    \item imports, toujours avec le noms de classes à importer, ordonnées java, swing, lib, locals.
    \item commentaire de description de la responsabilité de la classe.
    \item données membres, toujours private, sinon justifier.
    \item méthode, paramètres, indiquer toutes les exceptions que la méthode
    peut jeter.
    \item méthode requires, modifies, effets. requires indique les
    contraintes possibles; modifies indique toutes les entrées
    modifiées; effets définit le comportement.
    \item méthode à redéfinir dans la plus part de classes \pro{toString}.
    \item class de tests de toutes les méthodes.

\end{enumerate}

\chapter*{Introduction}
Ce document présente comment dans le logiciel \diamant{} les
données résidant dans des fichiers sont mis dans la forme
d'objets, afin d'être traites à l'intérieur de l'application.
\diamant{} est un logiciel de construction d'horaires, dans lequel
le chargement de données se fait à partir de  fichiers de texte.
Ces fichiers contiennent des enregistrements qui seront
``transformés'' en objets manipulables par l'application.
Ces objets seront éventuellement modifiés dans l'application
(uniquement certains champs changent de valeur). Tous les objets
seront sauvgardés par l'application dans un seul fichier, ainsi
l'application peut continuer un travail en prenant la dernière
sauvgarde faite.

Ici, nous donnons les détails de formats de fichiers, fichiers originaux et
fichiers produits; avec des indications sur l'utilisation de
champs et les classes concernant cette partie de l'application ainsi que
la correspondance entre champs des enregistrements fichiers et données membres.

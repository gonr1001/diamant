\chapter{Introduction}

\dx{} est un logiciel servant à la construction d'horaires de
cours et d'examens. Le manuel de l'utilisateur présente~:

\begin{itemize}
    %\item Une description des menus, des boîtes de dialogue et des fenêtres;
    \item Une description du processus de la construction d'horaires;
    \item Un exemple d'utilisation de \dx{} pour la préparation d'un horaire d'examens;
    \item Un exemple d'utilisation de \dx{} pour la préparation d'un horaire de cours (horaire de
cycle);

\item Une description des menus;

\item Une description des boîtes de dialogue;

\item Une description des fenêtres;

\item Guide d'installation.
\end{itemize}

Il s'agit de la version de \diamant{} telle que disponible en
fevrier 2004.

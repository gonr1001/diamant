\chapter{Installation et mise à jour}

L'installation et la mise à jour de \dx{} se font à partir du site web \emph{http://gap2.gel.usherb.ca/diamantbug} de la manière suivante:

\begin{description}
    \item[Authentification: ] Elle consiste à fournir un nom d'utilisateur et un mot de passe valide pour avoir accès au site web.
    \item[Accès à la page de téléchargement:] Après la validation de l'accès, cliquer sur le lien \emph{Accès en tant que Demandeur} pour faire apparaître une page avec la liste des opérations possibles \corrolivier{J'ai vérifié avec Domingo : les utilisateurs n'ont pas à cliquer sur un lien Accès en tant que Demandeur}. Cliquer sur le lien \emph{Téléchargements} pour faire apparaître la page de téléchargement à partir de laquelle peuvent se faire l'installation et la mise à jour de \dx{}.
\end{description}

\section{Installation}

Avant d'installer \dx{} il est nécessaire de s'assurer que la
\verb!version 1.5.0_01! ou ultérieure de la machine virtuelle Java
est installée sur votre ordinateur. Pour cela, il faut ouvrir
l'application \verb!Command DOS! et lancer la commande
\verb!java-version! comme illustrée sur la figure \ref{install}

 \begin{figure}[h]
      % Requires \usepackage{graphicx}
      \begin{center}
        \includegraphics[width=5.0in]{UserManualInputs/images/install.eps}
        \caption{Vérification de la version de la machine virtuelle Java}\label{install}
      \end{center}
 \end{figure}

S'il n'y a pas de machine virtuelle Java installée sur l'ordinateur,
il faut télécharger la dernière version disponible sur la page de
téléchargement en cliquant sur le lien \verb!j2re-1.5.0_01.exe! et
effectuer l'installation.

Si la machine virtuelle Java installée sur l'ordinateur est
antérieure à la \verb!version 1.5.0_01!, il faut d'abord
désinstaller l'ancienne version, puis télécharger la plus récente
disponible sur la page de téléchargement en cliquant sur le lien
\verb!j2re-1.5.0_01.exe! et enfin effectuer l'installation.

Une fois assuré de la bonne version de la machine virtuelle Java, l'installation de \dx{} peut commencer.

\begin{enumerate}
    \item Télécharger \dx{} en cliquant sur \verb!tictac.zip! à partir de la page de téléchargement puis ouvrir le fichier ;
    \item Extraire le contenu du fichier \verb!tictac.zip! vers l'emplacement de votre choix (sur votre disque, \verb!C:\! par exemple).
\end{enumerate}

L'installation étant terminée, vous pouvez commencer à utiliser
\dx{}. Pour lancer l'application, double-cliquez sur le fichier
\verb!Diamant.bat! se trouvant dans le dossier d'installation \dx{}
(c'est-à-dire l'emplacement choisi pour le contenu du fichier
\verb!tictac.zip!).

\section{Mise à jour}

La mise à jour de \dx{} peut se faire de deux façons:
\begin{itemize}
    \item Si la nouvelle version disponible sur la page de téléchargement du site web est un fichier \verb!.zip!:
    \begin{enumerate}
            \item Télécharger \dx{} en cliquant sur \verb!tictac.zip! à partir de la page de téléchargement puis ouvrir le fichier;
            \item Extraire le contenu du fichier \verb!tictac.zip! dans le dossier d'installation de \dx{}.
    \end{enumerate}

    \item Si la nouvelle version disponible sur la page de téléchargement du site web est le fichier \verb!tictac.jar!, téléchargez ce fichier en cliquant sur \verb!tictac.jar! et enregistrez-le dans le dossier d'installation de \dx{} (dans notre exemple, il s'agit du dossier \verb!C:\!\dx{}).
\end{itemize}

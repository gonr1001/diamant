\chapter{Installation et mise à jour}

L'installation et la mise à jour de \dx{} se font à partir du site web \emph{http://gap2.gel.usherb.ca:8080/diamantbug} de la manière suivante:

\begin{description}
    \item[Authentification: ] elle consiste à fournir un nom d'utilisateur et un mot de passe valide pour avoir accès au site web.
    \item[Accès à la page de téléchargement:] Après la validation de l'accès, cliquer sur le lien \emph{Accès en tant que Demandeur} pour faire apparaître une page avec la liste des opérations possibles. Cliquer sur le lien \emph{Téléchargements} pour faire apparaître la page de téléchargement à partir de laquelle peuvent se faire l'installation et la mise à jour de \dx{}.
\end{description}

\section{Installation}

Avant d'installer \dx{} il est nécessaire de s'assurer que la \verb! version 1.4.2_02! ou ultérieure de la machine virtuelle Java est installée sur votre ordinateur. Pour vérifier cela, ouvrir l'application \verb!Command DOS! et lancer la commande \verb!java -version! comme illustrée sur la figure \ref{install}

 \begin{figure}[h]
      % Requires \usepackage{graphicx}
      \begin{center}
        \includegraphics[width=5.0in]{UserManualInputs/images/install.eps}
        \caption{Vérification de la version de la machine virtuelle Java}\label{install}
      \end{center}
 \end{figure}
 
 S'il n'y a pas de machine virtuelle Java installée sur l'ordinateur, télécharger la version disponible sur la page de téléchargement en cliquant sur le lien \verb!j2re-1_4_2_04.exe! et installer là ensuite.

Si la version de la machine virtuelle Java installée sur l'ordinateur est antérieure à la \verb!version 1.4.2_02!, la désinstaller d'abord, télécharger ensuite la version disponible sur la page de téléchargement en cliquant sur le lien \verb!j2re-1_4_2_04.exe! et installer là enfin.
 
Une fois assuré de la bonne version de la machine virtuelle Java, l'installation de \dx{} peut commencer.

\begin{enumerate}
    \item Créer un dossier dans lequel sera installé \dx{} (Exemple: créer un dossier du nom de \dx{} sur le disque C:);
    \item Télécharger \dx{} en cliquant sur \verb!tictac.zip! à partir de la page de téléchargement et l'ouvrir ensuite;
    \item Extraire le contenu du fichier \verb!tictac.zip! dans le dossier créé en 1. (Dans notre exemple, il s'agit du dossier \verb!C:\!\dx{});
\end{enumerate}    

L'installation étant terminée, vous pouvez commencer à utiliser \dx{}. Pour le lancer double-cliquer sur le fichier \verb!Diamant.bat! se trouvant dans le dossier d'installation de \dx{}.

\section{Mise à jour}

La mise à jour de \dx{} peut se faire de deux façons:
\begin{itemize}
    \item Si la nouvelle version disponible sur la page de téléchargement du site web est un fichier \verb!.zip!:
    \begin{enumerate}
            \item Télécharger \dx{} en cliquant sur \verb!tictac.zip! à partir de la page de téléchargement et l'ouvrir ensuite;
            \item Extraire le contenu du fichier \verb!tictac.zip! dans le dossier d'installation de \dx{} (Dans notre exemple, il s'agit du dossier \verb!C:\!\dx{}).
    \end{enumerate}    
    
    \item Si la nouvelle version disponible sur la page de téléchargement du site web est le fichier \verb!tictac.jar!, Télécharger ce fichier en cliquant sur \verb!tictac.jar! et enregistrer le dans le dossier d'installation de \dx{} (Dans notre exemple, il s'agit du dossier \verb!C:\!\dx{}).
\end{itemize}

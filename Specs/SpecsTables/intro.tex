\section*{Introduction}Ce document est un essai pour définir les spécificationsfonctionnelles de \diamant{}. Étant donnée que le logiciel estdéjà construit, mais quelques fonctionnalités manquent, ledocument est orienté sur les nouvelles fonctionnalités.Les spécifications sont faites en s'inspirant de tables Tabularrepresentations de Parnas, en utilisant les documents de\cite{wilder95} et \cite{heninger80}.
